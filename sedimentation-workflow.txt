Sedimentation Workflow

**** Mesh Generation

- input: simulationID, dim, ncellx, ncelly, ncellz, xmin, ymin, zmin, xmax, ymax, zmax, ref_interval
- output: simulationID, r_fraction, c_fraction, max_h_level, hlevels

**** Create Equation Systems

- output: simulationID, Reynolds, Gr, Sc, Us, Diffusivity, xlock, fopc, theta, ex, ey, ez, c_factor

**** Get Maximum Iterations 

- output: simulationID, dt,tmax,n_time_steps,n_nonlinear_steps,nonlinear_tolerance,max_linear_iters,max_r_steps,write_interval, xdmf

**** Init Data Extraction

- output: simulationID, t_step, xdmf, u, v, w, p, s, d, Points0, Points1, Points2

**** Solver Simulation to the Fluid

- output: simulationID, t_step,transport_system.time,r,l,n_linear_iterations,final_linear_residual,norm_delta,norm_delta/u_norm,converged

**** Solver Simulation to the Sediments

- output: simulationID, t_step,transport_system.time,r,l,n_linear_iterations,final_linear_residual,norm_delta,norm_delta/u_norm,converged

**** Mesh Refinements

- output: simulationID, first_step_refinement, t_step, before_n_active_elem, after_n_active_elem

**** Mesh Writer

- output: simulationID, t_step, xdmf

**** (1) Data Extraction or (2) Mesh Statistics

- output: simulationID, t_step, xdmf, u, v, w, p, s, d, Points0, Points1, Points2

- output: simulationID, t_step, xdmf, sediments_amount

**** Mesh Aggregator

- output: simulationID, xdmf, n_processors
